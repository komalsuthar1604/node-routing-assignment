## to install
  npm init 
## to install nodemon 
  npm i nodemon 
## to run the application 
  npm run dev 

## Project Structure
The server targets and delivers files directly from the root working directory:

index.js — Core backend server configuration and routing logic.

home.html — Base land route webpage.

about.html — Informational bio route webpage.

contact.html — Communications route webpage.

404not.html — Custom error landing layout.

style.css — Global standalone layout configurations.
